Information on Modal

id - a unique identifier that is added to the modal window
Add: modal

data-target - used to indicate buttons on which modal window it refers
Add: button

modal-trigger - used to indicate that the button opens a modal window
Add: button

modal-close - used to indicate that the button closes the modal window
Add: button

modal-header - class for specifying the window heading
Add: modal

modal-main - class for specifying the main content of the window
Add: modal

modal-footer - class for specifying the footer of the window
Add: modal
